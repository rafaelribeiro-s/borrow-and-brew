# Borrow & Brew Architecture

## Objective

Learn Rust through the progressive building of a realistic coffee shop system.

## Structure

Borrow & Brew

|

├── Customer Service

├── Sales

├── Operations

├── Inventory

├── Reports

└── Administration

## Organization

The project is organized as a Cargo Workspace.

Each exercise represents an independent module in the operational evolution of the coffee shop.

## Example

Exercise 01 — Customer Registration (Shadowing)

Exercise 02 — Order Registration (Mutability)

Exercise 03 — Business Configuration (Constants)

Exercise 04 — Loyalty Program (If / Else)

Exercise 05 — Checkout Service (Functions)
